MORPH Runtime Core v1.0
Runtime State Persistence + Resume

This project is an open prototype focused on runtime state persistence, route memory, residue recovery, deferred work recovery, and resume orchestration.

Main ideas included in this version:
- state persistence to JSON
- resume from saved state
- restored route memory
- restored residue registry
- restored deferred work registry
- restored fabric state snapshot
- restored schedule snapshot
- simple runtime state manager
- orchestrated runtime flow for task routing and continuation
